# Smartflix

Smart Academy project. <br>
Technologies: Ruby 3.0, Rails 6.1, PostgreSQL. <br>

## Getting started
To get started, clone the repo and then install the needed gems:
```
$ bundle install
```
Next, migrate the database:
```
$ rails db:migrate
```
You can run this app in a local server:
```
$ rails server
```
